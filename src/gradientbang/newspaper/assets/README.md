# Gradient News & Observer Assets

These files are durable renderer inputs for the newspaper tools:

- `layout/front-page-04-geometry.json` defines the fixed region geometry.
- `references/full-page-style-fri-sat-pt.png` is the default generated full-page style reference.
- `references/masthead-style-wed-thu-pt.png` is used only for header/masthead generation.
- `references/gradient-bang-ui.png` is the Gradient Bang UI style reference.

The original Geek Club Gazette image is licensed for local reference use only and must not be committed or redistributed.
